# VB6 Outline

Extensão VS Code que fornece suporte ao **Outline** (visualização de símbolos) para arquivos Visual Basic 6.

## Funcionalidades

- ✅ Exibe símbolos de arquivos VB6 (.cls, .bas, .frm) no painel Outline
- ✅ Reconhece:
  - Classes e módulos
  - Funções (Public/Private/Friend Function)
  - Sub-rotinas (Public/Private/Friend Sub)
  - Propriedades (Property Get/Let/Set)
  - Types (estruturas)
  - Enums
  - Constantes (Const)
  - Variáveis (Dim, Public, Private)

## Como Usar

1. Abra um arquivo VB6 (.cls, .bas ou .frm)
2. O painel **Outline** (Explorador > Outline) exibirá automaticamente todos os símbolos
3. Clique em qualquer símbolo para navegar até sua definição

## Instalação (Desenvolvimento)

1. Clone este repositório
2. Execute `npm install`
3. Pressione F5 para abrir uma nova janela do VS Code com a extensão carregada
4. Abra um arquivo VB6 para testar

## Requisitos

- VS Code 1.108.1 ou superior

## Notas

Esta extensão foi desenvolvida para melhorar a experiência de desenvolvimento com código legado VB6 no VS Code.

**Enjoy!**
